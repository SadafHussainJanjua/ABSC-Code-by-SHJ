This project requires the installation of external libraries; to install them
you must write from the console or terminal:

pip install -r requirements.txt

In addition, you must download the following file to your computer:

https://nlp.stanford.edu/software/stanford-parser-full-2018-10-17.zip

When decompressing, within this folder you should search for the files
stanford-parser.jar and stanford-parser-3.9.2-models.jar and place them in the
project data folder.
